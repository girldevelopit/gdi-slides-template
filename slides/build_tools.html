Build tools

These are the tools that help us automate repetitive tasks:

  • Concatenating and minifying scripts
  • Pre-processing css
  • Optimizing images
  • Move minified files into a specific folder
  • Running tests
  • Live reload browser
  • Lint

Build tool options:

  • Gulp
  • Grunt
  • Broccoli
    • l
  • Brunch
  • Npm scripts
They all achieve similar goals, but for the purposes of this course we'll be using Gulp

Gulp

Unlike Grunt, Gulp uses nodejs streams and favors code over configuration Let's set up our first Gulp task

Installation


        $ npm install -g gulp
        # also install locally
        # --save-dev adds it as a dev dependency to our package.json
        $ npm install --save-dev gulp
        $ touch gulpfile.js

Open gulpfile.js in your text editor

Add the following lines:


        var gulp = require('gulp');

        gulp.task('default', function(){
            console.log('Our first gulp task')
        }
In your terminal run: 

        $ gulp

Commit!
