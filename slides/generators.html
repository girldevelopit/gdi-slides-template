Yeoman

Tool for generating new web projects

We are going to use one to quickly generate our project


Installation

  1. Navigate to www.yeoman.io
  2. Run 
    $ npm install -g yo
    to install with npm
  3. There are currently lot of generators available
We will use a very minimal one to scaffold a very basic website generator

Let's generate a project for our website today

  1. Generator we'll use for this class is Learn Front End
  2. Install our generator: 
    npm install -g generator-learnfe
  3. Create a new directory for our project (~/sites/marsbq -recommended) and cd into it, generate! 
    
            $mkdir my-new-project && cd $_
            yo learnfe
  4. Marvel at our new scaffolded project!
