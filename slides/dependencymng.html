Using dependency package management

The two most popular ones are Bower and Npm

We'll be using NPM for this class.

  1. Cd into your project dir
  2. Run: 
    
            $ npm init
  3. If you run `ls` command you will notice package.json file.
  4. you will also notive node_modules directory
