Let's make gulp task do something

Let's start with browsersync

  • browsersync.io
  • Starts a static web server
  • Automatically reloads the page when a specified file is changed
  • Can sync across browser and devices

Install and configure browsersync

$ npm install browser-sync --save-dev

    #Require it in your gulpfile
    var browserSync = require('browser-sync').create();
                    ....

    gulp.task('default',  function() {
        browserSync.init({
            server: './'
        });
        gulp.watch('src/**/*', browserSync.reload);
        gulp.watch('*.html', browserSync.reload);
 });
