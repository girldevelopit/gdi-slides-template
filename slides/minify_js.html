Let's also add javascript minification and image optimization

  • gulp-uglify
  • gulp-imagemin

You should be pros at this now


        $ npm install --save-dev gulp-uglify
        $ npm install --save-dev gulp-imagemin

        #All in one
        $ npm install --save-dev gulp-imagemin gulp-uglify

first, require them in your gulp file

Add 2 tasks to gulpfile.js called 'images' and 'scripts'


        var uglify =require('gulp-uglify')
        var imagemin =require('gulp-imangemin')

      gulp.task('scripts', function(){
		gulp.src(['src/scripts/**/*.js']) //source for gulp to watch
	       .pipe(sourcemaps.init())	//init sourcemaps
		   .pipe(uglify()) //stream! Run minify on the src
		   .pipe(sourcemaps.write())
		   .pipe(gulp.dest('dist/scripts')) //save to dist dir
});

..And images

Should be pretty similar to 'scripts'


       gulp.task('images', function(){
    		gulp.src(['src/img/**/*]) //source for gulp to watch
	       .pipe(imagemin())
		   .pipe(gulp.dest('dist/img')) //save to dist dir
		   .pipe(browserSync.stream()); //and finally reload

       });
Finally set your index.html, point to the new location and to watch
        
        gulp.watch('src/img/**/*, ['images]);

	    gulp.watch('src/scripts/**/*.js', ['scripts']);

Run gulp to see the results
