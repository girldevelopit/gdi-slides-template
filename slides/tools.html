Text Editor, Terminal and Browser

Editors:

We recommend Sublime or Atom for text editors

Download from https://www.sublimetext.com/3 or https://atom.io respectively

Terminal:

Mac: iTerm2, download https://www.iterm2.com/

Windows: Download http://cmder.net

Browsers:

Any modern browser: Firefox, Chrome, Safari

We will user Google Chrome for this class

Let's take a minute to make sure those are all setup

NodeJs

Installing NodeJs:

  1. Visit nodejs.org and click 'Install button'
  2. Verify it was installed by typing ```node -v``` in the terminal
  3. Verify npm was installed by typing ```npm -v``` in the terminal
        
       ```
~ node -v
v6.3.1
~ npm -v
3.10.3
        ```
Nodejs is a cross platform environment, really useful, even if you do not plan to write server side code.
